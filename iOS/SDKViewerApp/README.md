# SDK Viewer Mobile (iOS)

This module adds a mobile-first iOS GUI viewer for Unreal SDK dumps generated by this repository's iOS dumper.

## Workflow implemented

1. **First window (project setup):**
   - Create a new local project (saved under the app's Documents directory in `SDKViewerProjects/<project-name>`), or
   - Load an existing project.
2. **Import step:**
   - Import the iOS dumper output ZIP package.
   - ZIP is copied into the selected project directory, extracted locally, and parsed.
3. **Navigation step:**
   - Browse parsed packages and their structs/classes/enums.
   - View full parsed type source text.
4. **Developer helper:**
   - Multi-level pointer-chain calculator that generates C++ pointer resolution code.

## Parsed dump inputs

The importer expects these files from the ZIP package:
- `AIOHeader.hpp` (required)
- `script.json` (optional)

## Project generation (Xcode)

This app uses **XcodeGen** with `project.yml`:

```bash
cd iOS/SDKViewerApp
xcodegen generate
open SDKViewerMobile.xcodeproj
```

## Core module tests

The parsing and pointer-generation core is implemented in the Swift package target `SDKViewerApp`:

```bash
cd iOS/SDKViewerApp
swift test
```
